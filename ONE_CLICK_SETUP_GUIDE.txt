================================================================================
                           TROUBLESHOOTING
================================================================================

Q: "BUILD COMPLETE CHASE CITY" button does nothing?
A:
1. Make sure you have a Terrain in your scene
2. Check Console for errors
3. Try rebuilding in steps (1. City, then 2. Vehicles)

Q: Betting screen doesn't appear when pressing Play?
A:
1. Check that GameUI GameObject exists in scene
2. Check GameUIManager component has all references assigned
3. Check "Betting Canvas" is enabled in hierarchy

Q: Cars spawn inside buildings?
A:
1. City builder uses GetRandomStreetPosition() to avoid this
2. If it happens, rebuild with "2. Add Vehicles Only"

Q: No damage when crashing?
A:
1. Check PlayerCar has VehicleHealth component
2. Check PlayerDriverInput has collision code
3. Try hitting harder (damage threshold is 3.0 impulse)

Q: UI elements missing or wrong size?
A:
1. Check Canvas Scaler is set to "Scale With Screen Size"
2. Reference resolution should be 1920x1080
3. Rebuild UI by clicking "BUILD COMPLETE CHASE CITY" again

Q: Can't click buttons?
A:
1. Check EventSystem exists in scene
2. Check canvases have GraphicRaycaster component
3. Check buttons have Image component

Q: Money doesn't save between sessions?
A:
1. This is normal! PlayerPrefs saves automatically
2. To reset money: Edit â†’ Clear All PlayerPrefs
